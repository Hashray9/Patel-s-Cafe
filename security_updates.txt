=== PROJECT OVERVIEW & SECURITY POSTURE ===
Patel's Cafe PWA is a serverless frontend-only web application run entirely in the browser. The attack surface is limited to client-side vulnerabilities, package dependency safety, client-side input validation, and secure localStorage operations.
